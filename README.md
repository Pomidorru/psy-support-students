## Функциональность

### Публичная часть
- **Главная страница**: Информация о сервисе, разделы помощи
- **Инструменты самопомощи**: Описание доступных функций
- **FAQ**: Часто задаваемые вопросы
- **Контакты**: Форма обратной связи

### Личный кабинет (требует авторизации)
- **Мои сохранённые материалы**: Избранное с главной страницы
- **Аудиокниги**: Коллекция аудиоматериалов с плеером
- **Успокаивающая музыка**: Плейлисты для разных ситуаций
- **Дневник эмоций**: Записи настроения и заметки
- **Мой прогресс**: Статистика использования и график настроения

## Технологии

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Стилизация**: Собственные стили (mobile-first)
- **Графики**: Chart.js для визуализации прогресса
- **Хранение**: LocalStorage (для демо), Firebase (рекомендуется для продакшена)
- **Аудио**: HTML5 Audio API

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для тестирования личного кабинета используйте `dashboard.html`

## Разработка

### Добавление Firebase
Для полноценной работы замените LocalStorage на Firebase:

1. Создайте проект в Firebase Console
2. Добавьте Firebase SDK в HTML:
```html
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-auth.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-firestore.js"></script>
```

3. Инициализируйте Firebase в JavaScript файлах

### Аудио контент
- Замените placeholder аудио на реальные файлы
- Рассмотрите использование внешних сервисов (YouTube, Spotify) для embed
- Для больших файлов используйте облачное хранение
